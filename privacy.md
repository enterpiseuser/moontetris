# Kebijakan Privasi Moon Tetris

**Terakhir diperbarui:** 14 Maret 2025

## Pendahuluan

Moon Tetris ("kami", "aplikasi ini") menghargai privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami menangani informasi saat Anda menggunakan aplikasi Moon Tetris.

## Informasi yang Kami Kumpulkan

Moon Tetris adalah aplikasi WebView yang menyediakan akses ke game online. Aplikasi ini **tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda**.

### Data yang Tidak Kami Kumpulkan:
- Informasi identitas pribadi (nama, email, nomor telepon)
- Data lokasi
- Foto atau media
- Kontak
- Riwayat penelusuran
- Informasi keuangan atau pembayaran

## Koneksi Internet

Aplikasi ini memerlukan koneksi internet untuk mengakses game online yang di-host di `https://games.myshortlink.org/moon/`. Saat Anda menggunakan aplikasi:

- Aplikasi membuka website game melalui WebView
- Interaksi Anda dengan website game tunduk pada kebijakan privasi website tersebut
- Kami tidak memiliki kontrol atas data yang mungkin dikumpulkan oleh website pihak ketiga

## Cookie dan Data WebView

Aplikasi menggunakan WebView Android standar yang mungkin menyimpan:
- Cookie dari website game
- Cache lokal untuk performa yang lebih baik
- Data sesi untuk menjaga Anda tetap login (jika ada fitur login di website)

Data ini disimpan secara lokal di perangkat Anda dan dapat dihapus dengan menghapus data aplikasi melalui pengaturan Android.

## Izin Aplikasi

Aplikasi ini hanya meminta izin berikut:
- **INTERNET**: Untuk mengakses game online
- **ACCESS_NETWORK_STATE**: Untuk memeriksa koneksi internet

Kami **tidak** meminta izin untuk:
- Kamera
- Mikrofon
- Lokasi
- Penyimpanan eksternal
- Kontak

## Keamanan Data

Karena aplikasi ini tidak mengumpulkan atau menyimpan data pribadi Anda di server kami, tidak ada risiko data Anda bocor dari sistem kami. Semua data yang mungkin tersimpan hanya ada di perangkat Anda secara lokal.

## Layanan Pihak Ketiga

Kami tidak bertanggung jawab atas praktik privasi website pihak ketiga. Kami menyarankan Anda untuk membaca kebijakan privasi mereka.

## Privasi Anak-Anak

Aplikasi ini tidak secara khusus ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak. Jika Anda adalah orang tua atau wali dan mengetahui bahwa anak Anda memberikan informasi pribadi melalui website yang diakses melalui aplikasi ini, silakan hubungi kami.

### Data We Do NOT Collect:
- Personal identification information (name, email, phone number)
- Location data
- Photos or media
- Contacts
- Browsing history
- Financial or payment information

## Internet Connection

This application requires an internet connection to access online games hosted at `https://games.myshortlink.org/moon/`. When you use the application:

- The app opens the game website through WebView
- Your interaction with the game website is subject to that website's privacy policy
- We have no control over data that may be collected by third-party websites

## Cookies and WebView Data

The application uses standard Android WebView which may store:
- Cookies from the game website
- Local cache for better performance
- Session data to keep you logged in (if the website has login features)

This data is stored locally on your device and can be deleted by clearing the application data through Android settings.

## App Permissions

This application only requests the following permissions:
- **INTERNET**: To access online games

We do **NOT** request permissions for:
- Camera
